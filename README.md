## Installation

```bash
$ npm install commander-gulp-mailing
```


#### Command to Compile

```bash
$  commander-gulp-mailing mailing 'entry' --m 'ouput' 
```



#### Example

```bash
 "scripts": {
    "mailing": "commander-gulp-mailing mailing mailing 'test/templates/*.pug' 'test/**/*.pug' --m 'build/mailing' "
  }
```
